# Project Hosting Rule

- Do not present localhost, local servers, or temporary tunnel URLs as the final user-facing website.
- The public site must be deployed to a free online host with zero out-of-pocket cost for the user.
- Temporary local commands may be used only for development checks when unavoidable, and they must be stopped before handing work back.
- Prefer the existing static export path and free hosting such as Cloudflare Pages or Vercel Hobby.
